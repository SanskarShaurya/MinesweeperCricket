Game Information

Welcome to this crazy combination of Cricket and Minesweeper, fielders are spread randomly on the field and you need to pick blocks such that they don't contain fielders.

Different blocks have different scores, clicking on one may give you a boundary score or just a single. Clicking on blocks where fielders are present will make the game stop.

There are two game options present, you can either play alone or play with a friend in the Double Player mode. The double player mode introduces a turn based system where each player will pick a block each turn. The one with the highest score when both the players get out will win the game!

You can also choose the grid size of the game, there are 5 options available, from 6x6 to 10x10.

There are some hidden shields in the field, which will protect you if find a fielder in the next 3 turns, good luck on locating them!
